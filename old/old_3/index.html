Storitve

Žensko modno striženje

Žensko modno striženje se lahko razlikuje glede na trende, letni čas in osebne preference. Vendar pa obstaja nekaj splošnih značilnosti, ki so pogosto povezane s sodobnimi ženskimi striženji.

Moško striženje

Moško striženje je postopek striženja las pri moških, ki se izvaja s ciljem oblikovanja frizure, ki ustreza moškemu obrazu in slogu.

Otroško striženje

Pri otroškem striženju se frizer prilagaja otrokovim željam in ustvarimo različne pričeske, ki so prilagojene njihovemu slogu.
